# Tic-Tac-Toe Game

## Description
This is a simple tic-tac-toe game implemented in C. Two players take turns marking positions on a 3x3 board, and the game automatically checks for win and draw conditions.

## Features
- Add player move handling
- Display the game board
- Check win conditions for rows, columns, and diagonals
- Add draw condition to check for a tie game
- Improve code readability and structure"
